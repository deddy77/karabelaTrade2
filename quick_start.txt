# Trading Bot Azure VM Quick Start

## 1. Package the Bot
Run the packaging script:
```bash
chmod +x package_bot.sh
./package_bot.sh
```

This will create trading-bot-deploy.zip containing all necessary files.

## 2. Set Up Azure VM
- Create an Azure VM (Ubuntu recommended)
- For Windows: Ensure MetaTrader 5 is installed
- Open ports if needed for Discord notifications

## 3. Transfer Files
Upload trading-bot-deploy.zip to your VM using:
- SCP (Linux/macOS):
```bash
scp trading-bot-deploy.zip username@vm-ip-address:~/
```
- RDP (Windows): Copy via remote desktop

## 4. Deploy
Follow instructions in deployment_guide.txt on your VM:
```bash
unzip trading-bot-deploy.zip
cd trading-bot-deploy
cat deployment_guide.txt
```

## 5. Verify
Check the service is running:
- Linux: `sudo systemctl status trading-bot`
- Windows: Check Services manager
